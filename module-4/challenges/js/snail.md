## Prompt

Given an n x n array, return the array elements arranged from outermost elements to the middle element, traveling clockwise.

```js
const arrayMatrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
];

snail(arrayMatrix) #=> [1, 2, 3, 6, 9, 8, 7, 4, 5]
```
